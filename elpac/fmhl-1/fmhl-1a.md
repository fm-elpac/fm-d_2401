# FMHL-1a
胖蚊圣光机器 (1a 型)


## 主要结构

+ 整体: 细丝形机械臂, 直径 100um, 长度 5cm

+ 中央: 硅基芯支架, 宽 10um, 长 100um

+ 外皮: 塑料涂层

+ 电连接: 金线

+ 顶端操作台:

  - 可动导向脚: 可改变顶端前进方向

  - LED 照明灯

  - 微型光学显微镜

  - 微针

+ 外部辅助定位装置: 浸没式超声波


## 顶端显微摄像头

+ 像素数: 约 200x200

+ 单个像素大小: 0.2um

+ 像素阵列大小: 50um

+ 单色像素, 使用周围多色照明 LED

+ 照明 LED 大小: 5um
