+ 自动化组装

+ 自动化测试

+ 自动化维护/维修

## 远期目标

+ 实现能够完全制造自己的微型自举机器 (种子机器)

+ FMSP-1 **胖蚊火种计划**

  将种子机器播撒到别的星球, 实现一次发射, 长期运行 (自行更新)

  在别的星球上, 发芽 (展开), 生根 (扩大规模),
  开花 (同地球取得联系), 结果 (将新的种子发送至更多星球).

  先定一个小目标: 一亿年以内占领整个银河系.
