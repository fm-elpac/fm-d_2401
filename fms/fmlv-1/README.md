# FMLV-1
胖蚊低压电源系统


## 典型配置

4 条线路 (直流):

+ 普通 12V 1: 黄色, XT60 插头, 固定设备

+ 普通 12V 2: 黄色, XT60 插头, 固定设备

+ 普通 5V: 红色, XT60 插头, 小功率设备

+ 快充 12V: 黄色, XT90 插头


## 主要组件

+ 二手 ATX 电源 (12V, 5V 输出)

+ FMUPS-2 胖蚊小电源 (电池组)

+ 接口: 12V (XT60, XT90, XT30), 5V (USB-C)

+ 插座: XT60/4 口, XT60/6 口, 8 口

+ 导线 (外皮颜色采用 ATX 标准): 黄色 (12V), 红色 (5V)
