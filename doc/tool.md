# 技术/工具选型指导


## 基本原则

+ 应该使用开源的工具和平台

+ 建议使用开放标准

软件:

+ 建议基于 web 技术

+ 较高的开发效率

+ 能在廉价设备上运行

硬件:

+ 低成本

+ 安全, 无污染

+ 容易获得
